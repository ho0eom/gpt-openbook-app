문자열데이터 (ST)
◆문자열 데이터 (ST) 의개념
문자열 데이터는 후행 공백을 선택적으로 사용하는 좌측 정렬된 문자 세트 데이터
인쇄 가능한 ASCII 문자(0x20 ~ 0x7E)
문자열 데이터 안에 구분자를 포함하려면 취소 문자 사용 필요
텍스트데이터 (TX)
◆텍스트 데이터 (TX) 의개념
사용자 표시용 문자열 데이터로 선행 공백이 들어가며 좌측 정렬 아님
사용자에게 표시되는 명확성을 위해 선행 공백이 들어갈 수있음
•문자열 데이터와 차이점 : 선행 공백O, 좌측 정렬X
